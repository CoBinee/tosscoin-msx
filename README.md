## 説明
できるだけ報酬を得てください。<br>
こちらから遊べます。⇒  [WebMSX](http://webmsx.org/?MACHINE=MSX1J&PRESETS=SCC2&ROM=https://github.com/CoBinee/tosscoin-msx/raw/main/rom/TOSSCOIN.ROM)

## 操作
- `SPACE`<br>ジャンプ
- `←`, `→`<br>左右に移動（FREE RUN GAME のみ）
- ジョイスティックにも対応

## 動作環境
- 16KB 以上の RAM を搭載した MSX
